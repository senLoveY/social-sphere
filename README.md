# SocialSphere

---

**SocialSphere** — это учебный проект концепта современной социальной сети, ориентированный на отработку навыков чистой верстки и интеграции базовой логики на JavaScript.

## 🛠 Технологии

В проекте использовались следующие инструменты и технологии:

*   **HTML5** — семантическая разметка.
*   **CSS3** — стилизация (`css/styles.css` и модули в `css/`).
*   **JavaScript (ES6+)** — интерактивность и логика (файл `script.js`).
*   **Prettier** — для форматирования кода и поддержания единого стиля.
*   **npm** — управление зависимостями проекта.

## 📁 Структура проекта

```text
├── assets/             # Дополнительные ресурсы
├── css/                # Стили страниц
├── design/             # Ссылка на дизайн и документация макета
├── images/             # Графические ресурсы (иконки, фото)
├── js/                 # Скрипты JavaScript
├── index.html          # Главная страница проекта
└── package.json        # Конфигурация проекта и скрипты
```

## 🔧 Как запустить проект

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ваш-логин/название-проекта.git
    ```
2.  **Перейдите в папку проекта:**
    ```bash
    cd название-проекта
    ```
3.  **Установите зависимости (если необходимо):**
    ```bash
    npm install
    ```
4.  **Откройте `index.html` в браузере** (или используйте Live Server в VS Code).
